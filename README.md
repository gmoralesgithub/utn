# utn

#### how to setup

1. clone repo
```
<b>git clone ...</b>
```

2. cd into it
```
<b>cd utn</b>
```

3. install dependencies
```
<b>npm install && bower install</b>
```

4. add a platform (android, ios, or both)
```
<b>ionic platform add ios</b>
<b>ionic platform add android</b>
```

5. build
```
<b>ionic build ios</b>
```

6. emulate
```
<b>ionic emulate ios</b>
```
